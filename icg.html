WHAT IS "3 ADDRESS" / 3AC?

Three-address code represents instructions where each instruction has at most three operands. It's called "three-address" because it typically contains three addresses: one for the result and two for the operands. In this representation, temporary variables (like t1, t2) are often used to store intermediate values. Three-address code is simpler to analyze and optimize compared to the original high-level code.

a = t1

t0 = c * d

t1 = b + t0

WHAT ARE "QUADRUPLES"?

Quadruples are a specific type of 3AC where each instruction consists of 4 fields. They are: operator, 2 operands and a result

(*, c, d, t0)

(+, b, t0, t1)

WHAT IS "TRIPLES"?

It is similar to the Quadruples, but with just 3 fields, which is similar to the base 3AC. The fields are: operator, and the 2 operands

[*, c, d, t0]

[+, b, t0, t1]
